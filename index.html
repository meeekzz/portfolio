Clients Webpage

I recently built a website for a local business in Statesville, NC. This was a website that was built on a tight budget so I used a template and altered it to the clients liking. The website is still being developed as we speak, so it is not 100% complete. I will being adding an image carousel to each indiviudal house page so the images will look cleaner than they are currently presented. This was built off a bootstrap template, editing the HTML, CSS, and adding CSS Grid / Flexbox technology to build the indivdual house pages.


JS Calculator

I built this basic calculator for a school project to receive my Front End Libraries Certification. The calculator is built from scratch with HTML, CSS, and Vanilla Javascript while using some ES6 technology.


React Drum App

This project was also built for a school project for my Front End Technologies Certification. I built this in React to show my capabilities with Javascript libraries. This project reaches out to AmazonAWS to pull short mp3 clips that are assigned to buttons on the drum machine. You can click on these various buttons to play the sounds, or you can use your keyboard to push the key that is assigned to each button.


React API App

I built this project to show my strengths with React, and accessing API's. It reaches out to the Unsplash API and pulls search results of images based off the current search term that is being used. I've also done styling using CSS grid.
